**Slime AI** is a personal AI assistant app built by MUTYINT.

## Features
- AI Chat powered by multiple providers
- Image Generation
- Voice Input / Speech-to-Text
- Web Search Integration
- Dark glassmorphic UI

## Install

📥 [Download APK v3.0](https://github.com/emmanuelerogian723-alt/SlimeAI/releases/download/v3.0/SlimeAI_v3.apk)

1. Download the APK above
2. Go to Settings → Security → Enable "Unknown Sources" or "Install unknown apps"
3. Open the APK file and tap Install

## App Details
- Package: `com.mutyint.slimeai`
- Min Android: 5.0 (API 21)
- Target Android: 14 (API 34)
- Signed: V1 + V2 + V3 signatures

## Source Code
This is an open-source WebView wrapper. The full AI platform source is in the `src/` directory.
